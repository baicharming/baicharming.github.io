使用Hexo+Github搭建个人博客

By baicharming

发表于 2017-06-24

Hexo 是一款基于Node.js、快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

GitHub 是一个面向开源及私有软件项目的托管平台，它可以托管各种git库，并提供一个web界面。

下面我将介绍Hexo+Github搭建个人博客的基本步骤，本人在是在Ubuntu16.04操作系统上进行的。

申请Github账号

我们用Github进行代码托管，必须有一个Github账号，如果你还没有，马上去注册一个，注册步骤如下：

  • 首先进入GitHub 官网
  • 然后在注册页填写你的用户名，邮箱，密码，最后单击“SIGN UP FOR GITHUB”就行了
  • 接着你可以选择是付费还是使用免费的（默认是免费的），点击“Finish sign up”就行了
  • 最后别忘了去邮箱点击验证链接，完成整个注册过程

创建Github仓库

有了Github账号之后，去创建Github仓库，Repository的名称格式为username.github.io，比如我的username是baicharming，则我的仓库名称为baicharming.github.io，这样以来，你就可以通过访问网址https://baicharming.github.io/去访问该仓库下的页面了，后面我们写的博客文章都会上传到该仓库里。

申请域名并绑定

如果觉得自己的域名https://username.github.io比较low，那么你可以去申请以.com，.cn，.me等结尾的域名，域名服务商有很多，国内有万网，美橙，DNSPod，国外有GoDaddy， NameCheap，我选择的是DNSPod，申请了一个.com结尾的域名baicharming.com。

申请成功之后，进行域名解析，添加一条记录，“主机记录”为@，“记录类型”为CNAME，“记录值”为baicharming.github.io，这条记录的功能是将域名baicharming.github.io以CNAME的形式解析成baicharming.com，前提是在blog博客目录下的source文件夹下新建一个CNAME文件，内容为你申请的域名，比如我的是baicharming.com。这样以来，你访问域名baicharming.github.io或者baicharming.com都将到达的博客主页。
