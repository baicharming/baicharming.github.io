Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。以下介绍几种Markdown基本语法：

标题

在 Markdown 下，如果一段文字被定义为标题，只要在这段文字前加 # 号即可，1至6个#号对应1至6级标题，如下图所示。

列表

熟悉 HTML 的同学肯定知道有序列表与无序列表的区别，在 Markdown 下，列表的显示只需要在文字前加上 - 或 * 即可变为无序列表，有序列表则直接在文字前加1. 2. 3. 符号和文字之间加上一个字符的空格，如下图所示。

表格

表格比较麻烦，但掌握技巧之后也是很easy的，比如在 Markdown 编辑器中输入以下内容：

| Header 1 | Header 2 | Header 3 |

| ———— | ———– | ———— |

| Division 1 | Division 2 | Division 3 |

| Division 1 | Division 2 | Division 3 |

| Division 1 | Division 2 | Division 3 |

在页面中呈现的表格如下图所示。

图片和链接

插入图片与插入链接的语法很像，区别在一个 ! 号。

区块引用

如果你需要引用一小段别处的句子，那么就要用引用的格式，在一段文字前加上 > 符号表示引用，占有自己独立空间，并且有缩进，如下图所示。
