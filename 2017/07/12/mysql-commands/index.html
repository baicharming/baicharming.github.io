MySQL Workbench

MySQL Workbench是一个数据库架构师、开发人员和数据库管理员统一的视觉工具。MySQL Workbench提供数据建模、SQL开发和综合管理工具，用于服务器配置、用户管理、备份等等。MySQL Workbench可在Windows、Linux和Mac OS X.上使用。详情请看https://www.mysql.com/products/workbench/
