安装Ubuntu系统之后，中文输入法是必须要安装的，下面我将介绍Ubuntu16.04安装搜狗输入法的步骤。

下载搜狗输入法

打开搜狗输入法Linux版的官网http://pinyin.sogou.com/linux/?r=pinyin，并下载你需要的版本，我这里选择64位版。

安装搜狗输入法

这里介绍两种安装搜狗输入法的方法，第一种是直接点击下载的deb包进入Ubuntu软件中心进行安装，如下图所示；第二种是通过命令安装，命令如下。

添加软件源

打开Ubuntu软件更新管理器，点击Other Software，再点击Add，在弹出框中输入ppa:fcitx-team/nightly，如下图所示。

然后点击Add Source，之后再点击Revert，在弹出的框中点击Reload，如下图所示，此时需要稍等片刻，系统在更新软件源。

安装fcitx

在安装fcitx之前先更新一下系统，然后安装fcitx，命令如下，在安装fcitx的过程中会自动安装一些配置工具，比如fcitx-config-gtk3等。也可以在Ubuntu软件中心搜索fcitx，然后安装与fcitx相关的软件。总之确保fcitx已安装之后才能进行下面步骤。

终端输入im-config

添加fcitx的键盘输入法系统，因为sogou是基于fcitx的，而系统默认的是iBus。在终端输入im-config，这时会出现一个对话框，点击OK，有一个对话框，点击Yes，你会看到下面的对话框。如果上面是fcitx，就不用管，直接关闭；如果不是，就修改上面的ibus为fcitx，点击OK即可。

终端输入fcitx-config-gtk3

在终端中输入fcitx-config-gtk3，出现对话框，点击对话框左下角的（+）按钮，弹出另一个对话框。然后，取消Only Show Current Language（很重要，否则不能找到刚安装过的搜狗输入法!）最后，在输入框中输入sogou，如下图所示，选中点击OK即可，添加完后将其放置到列表的最下方，若没有找到搜狗输入法则重启系统再试一次。

重启电脑

用sudo reboot命令重启电脑，此时搜狗输入法已经安装成功，用shift键进行中英文输入法切换。
